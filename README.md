## Om Portalen

IntelligentForce er en AI-drevet forretningsautomatiseringsplattform som hjelper bedrifter med å identifisere og eliminere ineffektivitet. Portalen tilbyr skreddersydde automatiseringsløsninger med transparent ROI og rask implementering.

**Live URL:** [https://intelligentforce.ai/](https://intelligentforce.ai/)

---

## Nøkkelfunksjoner

| Funksjon | Beskrivelse |
|---|---|
| **Business Health Check** | 5 raske spørsmål som identifiserer bedriftens største utfordringer |
| **ROI-kalkulator** | Se nøyaktig hvor mye du sparer før du forplikter deg |
| **ALEX AI-assistent** | Personlig AI-assistent som svarer på spørsmål 24/7 |
| **Tospråklig** | Støtter norsk og engelsk |
| **Implementering på 2 uker** | Rask oppsett uten lang opplæring |

---

## Slik Fungerer Det

1. **Diagnose** – Svar på 5 raske spørsmål om bedriften din
2. **Kalkuler** – Se nøyaktig hvor mye du vil spare
3. **Automatiser** – Vi implementerer løsningen på 2 uker

## ALEX – AI-assistenten

ALEX er IntelligentForce sin innebygde AI-assistent som:
- Svarer på e-poster automatisk
- Gir 24/7 support
- Lærer av bedriftens spesifikke behov
- Hjelper brukere med å oppdage automatiseringspotensial
